GREATER HEIGHTS VR

Public Facing VR Flight Experience
Internship @ NASA Armstrong Flight Research Center

The Greater Heights project is a Virtual Reality (VR) experience that aims to engage the public with the work that NASA does through interactive story telling and education. The experience started in 2020 and contains a story mode that allows the user to embody Neil Armstrong in an historic flight of the X-15.

overview

The Greater Heights VR development project is an intern-led project at NASA's Armstrong Flight Research Center. The experience is a story-driven flight simulation intended for public outreach and education. The narrative of the experience is intended to capture the imagination of users and take them on an immersive journey in the shoes of Neil Armstrong and his famous test flight of NASAâ€™s X-15 aircraft. The user will be able to pilot the aircraft themselves allowing them to experience the same thrill Neil Armstrong did from the comfort of the ground.

I worked on this project as an intern of Spring 2024 alongside one other intern at the time. My role mainly consisted of integrating hand controllers and hand tracking features to the experience. This includes being able to control the virtual joysticks and buttons in the aircraft cockpit and programming the related plane physics and event sequences. Other than that, I have worked on polishing multiple features and refactoring a lot of the code to be more scalable for future interns.

We used Godot as the core game engine to build the experience. In addition to the VR headset and controllers, the project utilizes a motion chair to provide haptic feedback to the users and simulate the physics of the aircraft.

contribution

As part of my intern work, I worked on several features including:

  • Refactored input controls to allow for multiple inputs (i.e. mouse + keyboard, physical joystick, VR controllers, hand tracking, autopilot)
  • Integrated hand controllers using HTC Vive Wand Controllers
  • Imported hands and animations
  • Built an autopilot mode as an option if users do not want to control the aircraft
  • Integrated HTC Vive Tracker puck to offset unpredictable skew of Yaw Motion Chair relative to the headset
  • Simulated aircraft gauges (i.e. attitude, angle of attack, roll rate)
  • Built several user settings (i.e. controller sensitivity, autopilot mode, freeflight mode, chair mode, scene selection)
  • Built several dev operator keys to help when demoing the project (i.e. abort experience, reorient headset)
  • Programmed the chair to be less severe and slowly zeroing the chair state when the experience ends
  • Programmed hand controller haptics to match the shaking of the chair
  • Polished the world environment by improving the sky horizon and fading in space and stars
  • Optimized and retextured models to be more performant
  • Rehauled office scene which served as the storytelling / embodiment aspect of the experience where users learn about Neil Armstrong and the historical context of the X-15 flight
  • Demoed our project at a booth at IMS (Indianapolis Motor Speedway) which held the NASA Solar Eclipse event
